¶ Getting Started Guide for Developers

If you're a app or core developer, or someone comfortable getting up and running from a git clone, this method is for you.

If you clone the GitLab repository, you will need to build a number of assets using grunt.

Please do NOT use the master branch of iCollege in production.

¶ Prerequisite

  1. JRE 7 or greater be installed.
  2. Ruby 2.0 be installed. How to install it?
  3. Sencha Cmd be installed, and included into your System Path. How to install it?
  4. PhantomJS 1.8.2 or greater. Installation instructions can be found here
  5. Python 2.6 or greater(Recommend 2.7.* over 3.*), and included into your System Path. How to install it?

¶ Quickstart:

  1. npm install -g grunt-cli
  2. npm install -g casperjs
  3. npm install
  4. grunt init (download frontend assets, sencha touch framework and put them in the right place)
  5. npm start

¶ Hint for Developers:

Use grunt dev while you develop iCollege, express will restart for modifications of server side code
